## 1. ls (Listar archivos y directorios):*

Primero, hablemos sobre el comando 'ls'. Este comando nos permite listar los archivos y directorios en el directorio actual. Por ejemplo:
```shell
$ ls
```
## 2. cd (Cambiar de directorio):*

Ahora, aprendamos sobre 'cd'. Este comando nos permite cambiar de directorio. Por ejemplo:
```shell
$ cd directorio_destino
```

## 3. pwd (Mostrar directorio actual):

Con 'pwd' podemos saber en qué directorio estamos trabajando en este momento. Por ejemplo:
```shell
$ pwd
```

## 4. mkdir (Crear un nuevo directorio):

Si quieres crear un nuevo directorio, puedes usar el comando 'mkdir'. Por ejemplo:
```shell
$ mkdir nuevo_directorio
```

## 5. rm (Eliminar archivos o directorios):

'rm' se utiliza para eliminar archivos o directorios. ¡Pero ten cuidado, esto es irreversible! Por ejemplo:
```shell
$ rm archivo_a_eliminar
```

## 6. cp (Copiar archivos o directorios):

¿Necesitas hacer una copia de un archivo o directorio? Utiliza 'cp'. Por ejemplo:
```shell
$ cp archivo_a_copiar destino
```

## 7. mv (Mover o renombrar archivos o directorios):

Con 'mv' puedes mover o renombrar archivos y directorios. Por ejemplo:
```shell
$ mv archivo_a_mover destino
```

## 8. touch (Crear un archivo vacío):

Si necesitas crear un archivo vacío, usa 'touch'. Por ejemplo:
```shell
$ touch nuevo_archivo.txt
```

## 9. cat (Mostrar contenido de un archivo):

'cat' nos permite ver el contenido de un archivo. Por ejemplo:
```shell
$ cat archivo_a_mostrar
```

## 10. nano (Editor de texto en la terminal):

Si necesitas editar un archivo en la terminal, puedes usar 'nano'. Por ejemplo:
```shell
$ nano archivo_a_editar
```

## 11. grep (Buscar patrones en archivos):

'grep' es muy útil para buscar patrones en archivos. Por ejemplo, si queremos buscar la palabra "ejemplo" en un archivo:
```shell
$ grep "ejemplo" archivo_a_buscar
```

## 12. man (Acceder a la documentación del sistema):

Finalmente, 'man' nos da acceso a la documentación del sistema. Por ejemplo, si queremos saber más sobre el comando 'ls':
```shell
$ man ls
```
